Здравствуйте! Я бот для повторений.
Здесь вы можете удобно повторять термины и формулы.
1. Для просмотра групп вопросов для повторения используйте команду "/groups".
2. Для того чтобы выбрать группу, напишите её id.
3. Напишите свой ответ, для того чтобы получить правильный ответ и следующий вопрос.
4. Чтобы выйти в главное меню, используйте команду "/return".
